🎵 OpenMusic v4.8 — 功能说明书

曜石玻璃潮流增强版 · PySide6 + pygame + numpy + mutagen + Pillow

一、基础信息

  • 格式 MP3 / WAV / FLAC / OGG / AAC / M4A
  • 系统 Windows (macOS/Linux 待适配)

二、音乐管理

1. 扫描曲库

  • 程序启动时自动扫描同目录及子文件夹内的音乐文件
  • 自动跳过 .cache、_internal 等系统文件夹

2. 歌手识别（三级智能）

  • 一级：读取 ID3 标签中的歌手信息
  • 二级：按父文件夹名识别（最可靠）
  • 三级：从文件名解析（歌名 - 歌手.mp3 格式）

3. 封面管理

  • 自动提取 MP3/FLAC 专辑封面
  • 缓存封面到 .cache 文件夹
  • 根据封面主色调动态调整界面氛围光

4. 拖拽添加

  • 从文件管理器拖拽音乐文件或文件夹到窗口

5. 拖拽排序

  • 歌曲列表支持拖拽调整播放顺序

6. 搜索

  • 按歌名和歌手实时过滤
  • 快捷键 Ctrl+F 聚焦搜索框

7. 排序

  • 循环切换：歌名↑↓ → 歌手↑↓ → 时长↑↓

三、播放控制

基本控制

  • 播放/暂停：底部圆形按钮 或 Space 键
  • 上一首：⏮ 按钮 或 ← 键
  • 下一首：⏭ 按钮 或 → 键

进度控制

  • 波形进度条，支持点击跳转和拖动
  • 显示当前时间 / 总时长

播放模式（4种）

  • 顺序播放 — 播完列表停止
  • 列表循环 — 播完重头开始
  • 单曲循环 — 单曲无限重复
  • 随机漫游 — 随机跳转

倍速播放

  • 1×（正常）→ 1.5× → 2× → 0.5×

睡眠定时

  • 30 / 60 / 90 分钟后自动停止播放

四、音效与可视化

10段均衡器

  • 31Hz ~ 16KHz，每段 ±12dB
  • 5种预设：Normal / Pop / Rock / Bass / Vocal

频谱可视化（5种模式）

  • Bars — 柱状频谱
  • Ring — 环形频谱
  • Wave — 波形
  • 3D Bars — 3D 柱状
  • Particles — 粒子星云

氛围光效

  • 背景径向渐变 + 噪点纹理
  • 播放时根据封面色调动态光晕
  • 粒子星云 + 爆裂 + 波纹效果

六、信息面板

  • 统计条：曲库数 / 歌手数 / 收藏数 / 总时长 / 已听时长
  • 焦点面板：Now Playing 封面 + 歌名 + 歌手 + 频谱模式
  • 迷你播放信息：列表上方快捷操作（收藏♥ / 歌词 / 播放模式）
  • 歌词面板：支持 .lrc 格式，自动加载同名字幕文件

七、界面操作

  • 迷你模式：右上角 ⏺ 缩小为仅控制栏
  • 悬浮提示：鼠标悬停显示功能说明浮窗
  • 灵动岛通知：操作反馈滑动动画
  • 播放指示：当前行呼吸光柱 + 跳动条动画
  • 歌曲行操作：悬停显示 收藏♥ / 添加＋ / 菜单⋮

八、数据持久化

  • 收藏列表、播放模式、已听时长自动保存
  • 文件位置：.cache/openmusic_state.json

九、主题风格

  • 曜石玻璃配色：深色层叠 + 电光青 + 珊瑚点缀
  • 毛玻璃效果：底部栏、统计条、搜索框半透明磨砂
  • 动态光效：播放时根据封面自动调整氛围色
